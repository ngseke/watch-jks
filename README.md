```

```


## Set up the bot

Create your new bot with [BotFather](https://t.me/BotFather)

```
/newbot
```

```
/setcommands
```

```
start - 🐟 Say hello
showrecent - 👀 Show recent products
subscribe - 🔔 Subscribe the bot
unsubscribe - 🔕 Unsubscribe the bot
```
